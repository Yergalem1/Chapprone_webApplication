# Chapprone_webApplication
```Clone a repo :git clone https://github.com/Yergalem1/Chapprone_webApplication
Login to your firebase account
firebase login
Go to the root folder and initialize your firebase project
     -firebase init
Follow the following  steps to initialize the Firebase project 
 - Which Firebase features do you want to set up for this directory: select functions
 - Use an existing project:”chapperone-6a910 (chapperone)”
 - What language would you like to use to write Cloud Functions? JavaScript  
 - Do you want to use ESLint to catch probable bugs and enforce style? No
 - File functions/package.json already exist. Overwrite? No
 - File functions/index.js already exist. Overwrite? No
 - File functions/.gitignore already exists. Overwrite? No
 - Do you want to install dependencies with npm now? Yes
Navigate to the functions folder 
   -Install all the dependencies: npm install
   -Create .env file and put MongoDB connection string
   - Run server: npm run serve 
Navigate to frontEnd/react_csv folder
- Install all the dependencies: npm install
- Start React app: npm start 
- Make sure to connect your mongdb
